## üì¶ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+ recommended)
- [pnpm](https://pnpm.io/installation)

### Install Dependencies

```bash
pnpm install
```

### Development Server

Start the development server:

```bash
pnpm dev
```

### Build for Production

Build the application for production:

```bash
pnpm build
```
